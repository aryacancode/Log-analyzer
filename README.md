# Log-analyzer

This project involves developing a Python tool specifically designed to analyze log files, with a focus on /var/log/auth.log. The tool's primary function is to extract, examine, and interpret log data to uncover valuable insights about system operations, security events, and potential anomalies. It identifies key information such as command usage, user authentication changes, and security alerts, providing a detailed analysis that enhances the understanding of system behavior and security posture.
